Functions

povray_box (p1, p2)
The box shape is define by two opposite corners.

Parameters

  • p1: near lower left corner
  • p2: far upper right corner

Usage:

povray_box(v(-5,-5,-5), v(5,5,5)) > box(5) == povray_box(v(-5,-5,-5), v(5,5,5))

Return value:

a box
povray_cone (c1, r1, c2, r2)
The cone shape is defined by the center and radius of each end.

Parameters

  • c1: center of first end
  • r1: radius of first end
  • c2: center of other end
  • r2: radius of other end

Usage:

povray_cone(v(20,20,20),3,v(0,0,0),0) > cone(20,5,30) == povray_cone(v(0,0,0), 20, v(0,0,30), 5)

Return value:

a cone
povray_cylinder (c1, c2, radius)

  • radius: of the cylinder

povray_cylinder(v(20,20,20),v(0,0,0),3) > cylinder(5,20) == povray_cylinder(v(0,0,0), v(0,0,20), 5)

Return value:

a cylinder
povray_round_box (p1, p2, edge)
The round box shape is defined by two opposite corners and radius of the edges.

Parameters

  • p1: near lower left corner
  • p2: far upper right corner
  • edge: radius of the edges

Usage:

povray_round_box(v(0,0,0), v(20,30,40), 2)

Return value:

a round box that will fit entirely within a box object with the same corner points
povray_round_cone2 (c1, r1, c2, r2)
Creates a cone with rounded edges from a cone and two spheres. The returning object will not fit entirely within a cone object with the same end points and radii because of the spherical caps. The end points are not used for the conical portion, but for the spheres, a suitable cone is then generated to smoothly join them.

Parameters

  • c1: center of first end
  • r1: radius of first end
  • c2: center of other end
  • r2: radius of other end

Usage:

povray_round_cone2(v(20,20,20),3,v(0,0,0),0) > cone(20,5,30) == povray_cone(v(0,0,0), 20, v(0,0,30), 5)

Return value:

a round cone
povray_round_cylinder (c1, c2, radius, edge_radius)
Creates a cylinder with rounded edges from cylinders and tori. The resulting object will fit entirely within a cylinder object with the same end points and radius.

Parameters

  • c1: center of first end
  • c2: center of other end
  • radius: of the cylinder
  • edge_radius: the radius of the edges of the cylinder

Usage:

povray_round_cylinder(v(20,20,20),v(0,0,0),3,1)

Return value:

a rounded cylinder
povray_torus (r_major, r_minor)
Creates a torus centered on v(0,0,0) with major radius and minor radius.

Parameters

  • r_major: the major radius
  • r_minor: the minor radius

Usage:

emit(torus(20, 5));

Return value:

a torus
povray_wire_box (p1, p2, edge)
The wire box shape is made of cylinders and spheres. It is defined by two opposite corners and radius of the wire.

Parameters

  • p1: near lower left corner
  • p2: far upper right corner
  • edge: radius of the edges

Usage:

povray_wire_box(v(0,0,0), v(20,30,40), 2)

Return value:

a wire-frame box that will fit entirely within a box object with the same corner points
